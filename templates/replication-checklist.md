# Replication Checklist

Use this checklist when creating a replication guide for a new startup.

## Phase 1: Research
- [ ] Identify the core problem
- [ ] Define target audience
- [ ] Analyze pricing model
- [ ] Map tech stack
- [ ] Identify key features (MVP vs full product)

## Phase 2: MVP Build
- [ ] Set up project structure
- [ ] Implement core feature
- [ ] Add authentication
- [ ] Set up payments
- [ ] Create landing page

## Phase 3: Launch
- [ ] Choose launch platform (Product Hunt, Hacker News, etc.)
- [ ] Prepare launch assets
- [ ] Write launch copy
- [ ] Set up analytics
- [ ] Plan follow-up content

## Phase 4: Marketing
- [ ] Set up SEO basics
- [ ] Create content strategy
- [ ] Identify distribution channels
- [ ] Set up email list
- [ ] Plan referral system

## Phase 5: Growth
- [ ] Analyze user feedback
- [ ] Optimize conversion funnel
- [ ] Add referral/affiliate program
- [ ] Explore partnerships
- [ ] Plan feature roadmap

## Growth Strategy

1. **Viral demos** — Users share apps they built
2. **Content marketing** — "How I built X in Y minutes"
3. **Community** — Showcase gallery
4. **Partnerships** — Integrations with platforms
5. **Word of mouth** — Happy users refer others

- AI-native (not no-code)
- Real code output (not templates)
- Web + mobile from single prompt
- Fast iteration cycle

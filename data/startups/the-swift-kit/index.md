## Problem

Building iOS apps from scratch requires setting up authentication, payments, onboarding, and other boilerplate every time. This wastes days of development time.

## Solution

The Swift Kit is a complete SwiftUI starter kit with auth, payments, AI, and onboarding built in. Clone the repo, paste API keys, and ship a production-ready app in under an hour.

## Target Audience

- **Primary:** Indie iOS developers
- **Secondary:** Freelancers, small studios
- **Use cases:** Rapid app prototyping, client projects, side projects

## Tech Stack

- **Frontend:** SwiftUI
- **Backend:** Supabase
- **Payments:** RevenueCat, Dodo Payments
- **AI:** OpenAI

## Key Features

- Pre-built onboarding flow
- Paywall components
- Supabase auth integration
- AI integration ready
- Production-ready code

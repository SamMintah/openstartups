# VoiceType

> Voice-to-text that actually works for professionals

**Website:** [voicetype.com](https://voicetype.com) | **Source:** Public information

---

## Problem

Professionals waste hours typing when they could be speaking. Existing voice-to-text tools struggle with technical terminology, punctuation, and formatting.

## Solution

VoiceType is an AI-powered voice-to-text tool designed for professionals. It understands technical jargon, adds proper punctuation, and formats text for different contexts.

## Target Audience

- **Primary:** Writers, journalists, content creators
- **Secondary:** Developers, lawyers, doctors
- **Use cases:** Blog posts, documentation, reports, emails

## Tech Stack

- **Frontend:** Web app
- **AI:** Whisper API + custom fine-tuning
- **Backend:** Node.js
- **Payments:** Stripe

## Key Features

- High-accuracy transcription
- Technical vocabulary support
- Smart punctuation
- Format templates
- Export to multiple formats

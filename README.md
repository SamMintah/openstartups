## What is this?

A free, community-driven collection of **real profitable startups** with detailed replication playbooks. Each entry includes:

- The problem it solves and who it's for
- Business model, pricing, and revenue data
- Full tech stack breakdown
- Step-by-step replication guide (from MVP to launch)
- Verified metrics from TrustMRR

**Goal:** Help indie hackers and bootstrappers learn from and replicate successful businesses.

## How to Use

1. **Browse** the `data/startups/` directory
2. **Read** the `index.md` for an overview
3. **Study** the `business-model.md` for pricing and revenue strategy
4. **Follow** the `replication-guide.md` to build your own version
5. **Check** `metrics.json` for structured data

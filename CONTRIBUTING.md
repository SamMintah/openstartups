# Contributing to Open Startups

Thank you for your interest in contributing! This project thrives on community contributions.

## How to Contribute

### Adding a New Startup

1. Fork the repository
2. Copy a template from `templates/`
3. Fill in the content following the existing format
4. Add entry to `data/index.json`
5. Submit a pull request

### Quality Standards

- **Verified data only** — Use TrustMRR or other verified sources
- **Accurate information** — Double-check all metrics and details
- **Actionable content** — Replication guides must have specific steps
- **Proper attribution** — Credit original sources

### Content Guidelines

- Be respectful of original founders
- Don't reveal proprietary information
- Focus on learning and inspiration
- Include disclaimers where required

### Technical Contributions

- Fix errors in existing entries
- Improve replication guides
- Add new categories
- Enhance templates

## Pull Request Process

1. Create a descriptive PR title
2. Fill out the PR template
3. Ensure all checks pass
4. Wait for review

## Questions?

Open an issue or start a discussion!
